每日一题1-石之游戏

[每日一题] (石之游戏)
n堆石头排成一行，你每次可以把相邻的2堆石头聚成1堆，聚合后的石头数会成为你的分数的一部分。
问当进行n-1次，所有石头聚成一堆后，怎么使得总分最小。
例子: [3, 4, 3]
  第一次 3+4=7 -> [7, 3]
  第二次 (7) + 7 + 3 = 17 -> [17]
  总分17最少。

Follow-up(Google onsite): 变成一个环

*解题思路*
暴力解法: 观察题目后发现题目的关键在于使得每次加出来的和最少。那么这道题是否符合贪心性质，也即每一步都选择当前的最优解(选当前和最小的两个相邻的数)，可以求得全局最优解呢？

答案是可以的，可以用数学归纳法证明

Base Case:
  首先假设n = 1, 此时当前必须选使得和最小的数，所以成立
  n = 2, 此时也必须选使得当前和最小的数，所以成立

Deduction:
   假设n = k-1, 使得和最小的选择符合贪心性质
   当n = k的时候，此时数组在做最后一步合并的时候，一定会选择两个堆n1和n2，而n1和n2都是由(1…n-1)个数组成的使当前分数最小的堆，因此n1和n2都是按照贪心选择生成的堆。而我们在合并这两个堆的时候也使用了一次贪心选择

所以我们可以得出这道题是符合贪心性质的
